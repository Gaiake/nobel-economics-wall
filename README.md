# 学科思政基地六屏大屏

浙江大学学科思政育人示范基地一楼触摸大屏项目。页面按 6 块竖屏拼接设计，真实画布尺寸为 `6480 × 1920`。

## 当前布局

- 第 1 屏：基地导航页，展示项目介绍，并切换中间两屏内容。
- 第 2 屏：诺贝尔经济学奖获奖者群像墙，支持按年代筛选。
- 第 3 屏：当前选中获奖者详情，展示获奖年份、理论贡献、名言与简介。
- 第 4-6 屏：同花顺宏观经济大屏，嵌入地址为 `https://board.10jqka.com.cn/fe/largescreen/macroEconomySchool.html`。

当点击“科普互动游戏”或“科普电子书橱”时，第 2-3 屏会切换为对应互动内容；5 分钟无触摸操作后自动回到诺奖展示。

## 功能

- 覆盖 `1969-2025` 年诺贝尔经济学奖得主。
- 诺奖展示采用第 2 屏群像、第 3 屏单人详情的物理分屏布局。
- 投教互动游戏包括“期货玩家”“红色证券史”“投资日记”。
- 电子书橱嵌入上海期货交易所电子书橱页面。
- 右三屏嵌入同花顺宏观经济大屏。
- 纯静态前端，无构建步骤，可直接部署到静态站点或 GitHub Pages。

## 本地预览

推荐使用项目内置预览脚本：

```bash
node scripts/serve-preview.mjs
```

默认地址：

```text
http://127.0.0.1:8080/
```

如需换端口：

```bash
PORT=8081 node scripts/serve-preview.mjs
```

页面是 `6480 × 1920` 真实画布，本地普通显示器预览时需要横向或纵向滚动查看完整六屏。

## 部署

本项目是静态页面，部署时保留以下文件和目录即可：

```text
index.html
src/
assets/
```

GitHub Pages 可直接部署仓库根目录。部署后建议在目标主机上使用浏览器全屏打开页面，并让窗口覆盖完整 `6480 × 1920` 输出区域。

## 测试

```bash
node --test tests/*.test.mjs
```

测试覆盖数据完整性、六屏布局常量、导航状态、嵌入链接和关键样式约束。

## 项目结构

```text
index.html              页面入口
src/app.js              页面渲染与交互逻辑
src/displayShell.js     六屏尺寸、导航项、外部嵌入链接配置
src/laureateState.js    年代与人物选择状态
src/styles.css          大屏布局与视觉样式
src/data/laureates.json 诺奖得主展示数据
assets/portraits/       人物头像
scripts/serve-preview.mjs 本地静态预览服务
tests/                  自动化测试
```
